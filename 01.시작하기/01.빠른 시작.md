# Vue 애플리케이션 만들기

## 설치

`npm init vue@latest`
`yarn create vue@latest`

or CDN 사용
`<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>`

## 개발 서버 시작

```bash
cd <your-project-name>

npm install

npm run dev
```
```jsx
cd <your-project-name>
yarn
yarn dev
```

## 배포

`npm run build`
`yarn build`
